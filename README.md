📚 MyAttendanceManager

A Flutter-based Attendance Management System that helps students track attendance, upload academic timetables, and manage schedules online.

🚀 Features

* User Registration & Login
* Attendance Tracking Dashboard
* PDF Timetable Upload
* Academic Calendar Management
* Cloud Database Storage (MongoDB Atlas)
* REST API Backend (Node.js + Express)
* Responsive Flutter Web Interface
* Online Deployment with Netlify and Render

🛠️ Tech Stack

Frontend

* Flutter Web
* Dart

Backend

* Node.js
* Express.js

Database

* MongoDB Atlas

Hosting

* Netlify (Frontend)
* Render (Backend)

⚙️ Installation

Clone Repository

git clone https://github.com/shaiksohelsiddiq5/MyAttendanceManager.git
cd MyAttendanceManager

Frontend

cd frontend
flutter pub get
flutter run

Backend

cd backend
npm install
node server.js
